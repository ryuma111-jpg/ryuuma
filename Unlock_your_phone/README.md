画像認証ログインシステム

画像を使ってログイン認証を行うWebアプリです。
登録された画像とアップロードされた画像を比較し、一致した場合のみログインできます。

🚀 デモ内容
画像をアップロード
登録画像と比較
一致した場合のみ「解除」ボタンが有効化
ログイン状態をセッションで管理
🛠️ 使用技術
Python
Streamlit
OpenCV（画像処理）
NumPy
Pillow
⚙️ 機能一覧
🔐 画像によるログイン認証
🖼️ 複数画像による認証対応
📊 一致度の表示（差分の合計値）
🔁 ログイン / ログアウト機能
📁 複数画像の読み込み（リスト管理）
🧠 仕組み
登録画像を複数読み込む
アップロードされた画像をリサイズ
各画像と差分（cv2.absdiff）を計算
差分の合計値が一定以下なら一致と判定
pip install streamlit opencv-python pillow numpy
streamlit run portfolio.py
⚠️ 注意点
画像サイズや明るさによって一致精度が変わる場合があります
完全なセキュリティ用途ではなく、学習目的の実装です

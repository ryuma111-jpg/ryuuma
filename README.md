# 画像一致判定プログラム
画像一致判定プログラム
StreamlitとOpenCVを使用して、2つの画像が一致するかチェックします。
## 概要
- 指定した画像（`peach.jpg`）と別の画像を比較します。
- 画像が完全に一致した場合、画面に「開く」と表示され、お祝いの風船（Balloons）が飛びます。
- 一致しない場合は「開かない」と表示されます。

## 実行方法
以下のコマンドでアプリケーションを起動できます。
```bash
pip install -r requirements.txt
streamlit run Portfolio.py
